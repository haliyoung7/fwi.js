    Content

    This is the content class it can be used in two ways. You can access the static methods if you just need a one off operation, or you can create a content item to execute multiple operations on it.

    Example:

    var item = new Content('Places_Live_Data', 'Live_Data_Region);

item.getDetails()

item.play()

    or

    var url_contents = Content.Url('https://example.com');
